<Section icon={Target} title="Step-by-Step Crossplay Connection Guide">

Connecting with friends across different platforms requires using the in-game lobby system rather than platform-specific friend lists. Since Steam invites do not work directly on Xbox consoles, players must rely on the universal Invite Code system. This ensures that any player on any platform can find and join the correct adventure. Follow these steps to host or join a session safely.

<Steps>
  <Step num="1" title="Select Multiplayer from the Main Menu">
    The host must launch For The King II and select **Multiplayer** from the main title screen. Choose the correct server region that is geographically closest to all players to minimize latency.
  </Step>
  <Step num="2" title="Create a Private or Unlisted Lobby">
    Click **Create Adventure**. Set the lobby privacy to **Unlisted** or **Private** and assign a secure password. This prevents random players from joining your game mid-lobby.
  </Step>
  <Step num="3" title="Locate and Share the Invite Code">
    Once inside the lobby, open the Multiplayer Menu. Under the **Details** tab, find the **Online Invite Code** field. Click the **Eye Icon** to reveal the unique code, and copy or read it to your friends.
  </Step>
  <Step num="4" title="Join via the Invite Code">
    The joining players on Steam or Xbox must go to the **Multiplayer** menu, select **Connect by Invite Code**, enter the shared code, and input the lobby password.
  </Step>
  <Step num="5" title="Assemble the Party and Start the Run">
    Ensure all players have joined the lobby and selected their characters *before* the host loads the save file or starts the journey. Players cannot join a campaign that is already in progress.
  </Step>
</Steps>

| Role | Action Required | Menu Location | Key Setting |
| :--- | :--- | :--- | :--- |
| **Host** | Create the server | Multiplayer > Create Adventure | Set to **Unlisted** |
| **Host** | Get connection key | Lobby Menu > Details | Click **Eye Icon** for code |
| **Guest** | Enter the server | Multiplayer > Connect by Code | Input **Invite Code** & password |
| **All**| Select characters | Character Selection Map | Complete *before* launch |

<Callout type="warning" title="Save File Lock-In">
Always make sure your entire group is present in the lobby before loading a saved campaign. If a player is missing when the host launches the map, they will be locked out of that session until the host returns to the main menu.
</Callout>

<Section icon={Shield} title="Understanding Progress, Saves, and Lore Store Unlocks">

While multiplayer co-op is completely cross-platform, progress tracking behaves differently. For The King II does not feature cross-progression or cross-save capabilities. The developer, IronOak Games, confirmed that as a smaller independent studio, managing cross-platform account databases and complying with varied legal standards for data collection was beyond their current infrastructure limits. This means your progression is strictly tied to the platform where you play.
