<Section icon={Target} title="Hosting and Joining For The King II Xbox PC Crossplay Lobbies">

Setting up a cross-platform session requires the host to properly configure the lobby parameters before launching the adventure. Because the game does not support dynamic mid-game joining, establishing a stable connection in the pre-game lobby is critical. If a player disconnects during a run, the host must save the game, exit to the main menu, and recreate the lobby to allow them back in.

Follow these structured steps to establish a stable crossplay session between Xbox and PC players.

<Steps>
  <Step num="1" title="Select Multiplayer Mode">
    The designated host must navigate to the main menu and select the Multiplayer option. Choose the option to host a new adventure rather than starting a local solo game.
  </Step>
  <Step num="2" title="Configure Lobby Settings">
    Set your matchmaking region to match your teammates. Create a unique Room Name and apply a secure password to prevent random players from occupying character slots.
  </Step>
  <Step num="3" title="Share Lobby Details">
    Provide your teammates with the exact Room Name, password, and chosen server region. Xbox and PC players can use the in-game server browser to search for the room name directly.
  </Step>
  <Step num="4" title="Assign Characters and Launch">
    Once all players have joined the lobby, distribute the party characters among the active players. The host can then safely load an existing save or start a fresh campaign chapter.
  </Step>
</Steps>

If a player experiences a connection failure during a long dungeon run, do not panic. Your progress is safe. The host simply needs to save the current state, quit to the main menu, re-host the lobby with the same settings, and let the disconnected player rejoin before reloading the active save file.

<Callout type="tip" title="Version Alignment Rule">
Ensure that all players have updated their games to the identical version. Because PC and console update certification pipelines sometimes differ, minor version mismatches can temporarily prevent crossplay lobbies from appearing in the server browser.
</Callout>
