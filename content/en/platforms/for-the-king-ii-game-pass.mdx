<Section icon={Users} title="How to Setup Multiplayer and Crossplay">

Setting up a session in For The King II on Game Pass is straightforward, but it requires specific steps to ensure all party members are synchronized. The game is built for cooperation, and while solo play is possible (controlling all four characters), the true experience shines when players split the roles.

<Steps>
  <Step num="1" title="Create a Multiplayer Lobby">
    The host must select 'Multiplayer' from the main menu and create a new adventure. You can set the room to Public, Friends Only, or Private with a password.
  </Step>
  <Step num="2" title="Invite Friends Across Platforms">
    Game Pass players can invite friends through the Xbox overlay (Win+G on PC). For crossplay with Steam users, provide the Room Name and ensure the 'Crossplay' toggle is enabled in the settings.
  </Step>
  <Step num="3" title="Assign Character Slots">
    Once all 2-4 players are in the lobby, the host assigns character slots. In a 2-player game, each player typically controls two characters to maintain a full 4-person party.
  </Step>
  <Step num="4" title="Finalize Loadouts and Start">
    Every player selects their class, starting trait, and initial items using Loadout Points. Once everyone clicks 'Ready,' the host can launch the campaign.
  </Step>
</Steps>

<Callout type="warning" title="Joining Rules">
Players **cannot join a game already in progress**. If a player disconnects, the host must save the game, return to the main menu, recreate the lobby, and have everyone rejoin before loading the save file.
</Callout>

<Section icon={Target} title="Combat Mechanics: Mastering the Battle Grid">

The Battle Grid is a new addition to the sequel, introducing a 2x4 tactical layout. Positioning determines who can be targeted and who can protect their allies. Understanding how to use the grid effectively is essential for surviving the harder chapters available on Game Pass.

- **Front Row**: Reserved for tanks like the Blacksmith. They can use the **Guard** ability to protect allies directly behind them.
- **Back Row**: Ideal for fragile casters and ranged attackers. Most melee enemies must clear the front row before reaching these targets.
- **Area of Effect (AOE)**: Some attacks hit entire rows or columns. Avoid clumping all characters in one vertical lane if the enemy has column-piercing attacks.
- **Focus Points**: Use Focus to guarantee successful rolls on attacks or skills. Never enter a boss fight with zero Focus.

| Status Effect | Description | Counter / Cure |
| :--- | :--- | :--- |
| **Dazed** | Removes secondary action (movement/items) | Panax / Wait 1 turn |
| **Stunned** | Skips the entire next turn | Salt / High Resistance |
| **Bleed** | Damage over time every turn | Bandages / Healing |
| **Acid** | Dissolves equipped gear over time | Sponge / Quick Kill |
| **Shock** | Next roll automatically fails | Panax / High Resistance |

<Callout type="info" title="Focus Restoration">
You can restore Focus by **Meditating** in towns, using **Golden Root**, or finding a **Stone Hero** on the overworld map. Use your Focus aggressively during difficult encounters, as a dead party cannot use saved resources.
</Callout>
